# Developer Tools v1.0.7

静态网页开发工具箱，支持中文 / English，支持 PC 和移动端，可部署到 GitHub Pages。

## 本版本更新

- 页面 UI 已回退到 v1.0.2 风格。
- 保留后续新增功能：版本号显示、PC favicon、iPhone Safari 添加到主屏幕图标、manifest 配置。
- 继续删除左上角“静态网页工具集”副标题。

## 当前工具

1. String Concatenation
   - 支持输入格式 + 输出格式互转。
   - 支持换行、逗号、单引号、双引号、无分隔格式。

2. Excel ⇔ JSON
   - Excel 转 JSON。
   - JSON 转 Excel。
   - 支持多 Sheet。

## 部署

上传整个项目到 GitHub Pages 即可。

当前使用 CDN：

- jQuery
- xlsx.full.min.js


## v1.0.7

- PC端改为固定工作台布局，默认隐藏浏览器滚动条。
- String Concatenation 输入/输出区重新调整为左右双编辑器布局。
- Excel ⇔ JSON 工具区重新调整为顶部参数区、操作区、主编辑区。
- 当内容较多时，由文本编辑区内部滚动，不优先出现整页滚动。
